陽明交大 Dreamhack 商業競賽 - 亞軍

• 任務 (Situation/Task)： 在 15 支專業參賽隊伍中競爭，設計一套「針
  對高層次學科知識能力或軟實力」的「教學或/及評量」的解決方案/產
  品服務  
• 行動 (Action)： 開發針對「臨床護理培訓」的數位化轉型方案。負責全
  流程數位孿生系統規劃，將複雜的護理流程轉化為可互動的虛擬模型。
  統籌團隊直接向創投創辦人進行 Pitch 提案，並制定從概念驗證到市場
  落地的完整 GTM Roadmap。 
• 結果 (Result)： 榮獲競賽亞軍，證明具備將前瞻技術（數位孿生）轉化
  為具商業價值之教育科技產品的能力。
